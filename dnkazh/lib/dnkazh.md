# DNKazh

## `getPort()`

Returns the port used by the API, default is `53`.

+ ### Returns

    1. `number`

## `setPort(_port)`

Sets the port used by the API, default is `53`

+ ### Parameters

    1. `number`

## `whodns()`

Requests the address of the DNS server via broadcast.

+ ### Returns

    1. `uuid` | `nil` The address of the server.
